# telegram_video_to_gif

Telegram-бот для конвертации видео и кружков в GIF. Принимает видеофайлы или видеосообщения (кружки), конвертирует их через FFmpeg и отправляет обратно в чат. Поддерживает настройку параметров конвертации для каждого пользователя и режим вайтлиста.

## Возможности

- Конвертация видеофайлов и видеосообщений (video note) в GIF
- Настройки пользователя: предельный размер GIF после конвертации в мегабайтах
- Белый список пользователей (whitelist) — бот работает только для разрешённых
- Панель администратора: управление вайтлистом, просмотр статистики
- Многоязычность через локали (`locales/`)
- Хранение настроек пользователей в SQLite через SQLAlchemy (async)
- Логирование через встроенный логгер

## Стек

- Python 3.10+
- [aiogram 3](https://docs.aiogram.dev/) — Telegram Bot API
- SQLAlchemy (asyncio) + aiosqlite — асинхронная база данных
- FFmpeg — конвертация видео в GIF
- python-dotenv — загрузка секретов из `.env`
- uvloop — ускорение event loop на Linux

## Структура

| Файл / Папка | Назначение |
|---|---|
| `app.py` | Точка входа, регистрация роутеров |
| `config.py` | Настройки из `.env` через dataclass |
| `handlers/` | Обработчики: старт, видео, настройки, админ, вайтлист, ошибки |
| `services/` | Бизнес-логика конвертации и работы с файлами |
| `db/` | Модели и репозитории базы данных |
| `core/` | Логгер и вспомогательные компоненты |
| `keyboards/` | Inline-клавиатуры |
| `locales/` | Файлы локализации |
| `utils/` | Вспомогательные утилиты |

## Запуск

1. Скопируйте `.env.example` в `.env` и заполните значения:

```bash
cp .env.example .env
```

2. Установите FFmpeg (если не установлен):

```bash
# Ubuntu / Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Запустите бота:

```bash
python app.py
```

## Переменные окружения

См. файл `.env.example` в корне репозитория.

| Переменная | Описание | Значение по умолчанию |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram-бота | — (обязательно) |
| `DATABASE_URL` | URL базы данных | `sqlite+aiosqlite:///./bot.db` |
| `DEFAULT_LANG` | Язык по умолчанию | `en` |
| `MAX_INPUT_MB` | Максимальный размер входного файла (МБ) | `20` |
| `TEMP_DIR` | Папка для временных файлов | `./tmp` |
| `FFMPEG_PATH` | Путь к исполняемому файлу FFmpeg | `ffmpeg` |
| `ADMINS` | ID администраторов через запятую | — |
